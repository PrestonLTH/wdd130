What is HTML?
Gives structure to a web page
Nesting = Tags inside of tags a.k.a parent tags and child tags
<h1> Information in between are Main Heading Tags
<h2> Sub-Headings
<p> Paragraph texts

H = Hyper
T = Text
M = Markup
L = Language

Hypertext allows users to jump from one page of information to another
Markup is the language being used to show the information
